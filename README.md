# Solana GDM Project

This project implements Liquefied Petroleum Gas (LPG) distribution using Solana blockchain.

## Components

* GDM Allocation Engine
* Solana Smart Contract
* Odoo Integration
* Dashboard

## Workflow

Odoo → Allocation Engine → Smart Contract → Distribution → Dashboard

## Tech Stack

* Solana Blockchain
* Python AI Engine
* Odoo API
* Dashboard
